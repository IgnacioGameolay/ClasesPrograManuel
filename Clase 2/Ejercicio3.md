## Ejercicio 3 — Vida Real (Básico)

Se desea modelar un sistema de transporte.

Se sabe que todos los vehículos tienen una marca y una velocidad.

Se pide:

1. Definir una clase llamada `Vehiculo` que contenga:
   - Un atributo `marca` de tipo String
   - Un atributo `velocidad` de tipo int

2. Implementar un constructor que inicialice ambos atributos.

3. Implementar un método llamado `acelerar(int incremento)` que aumente la velocidad.

4. Definir una subclase llamada `Auto` que:
   - Herede de `Vehiculo`
   - Tenga un atributo `puertas` de tipo int

5. Implementar un constructor para la clase `Auto`.

6. En el método `main`, crear un objeto de tipo `Auto` y mostrar su información.
